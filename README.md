# Xeirographa

Guided lessons for learning to read Greek manuscripts

Available for use at [xeirographa.com](www.xeirographa.com)

## Running and Editing Locally

### Requirements

1. Node package manager for installations
2. Typescript
3. React 18

### Installation

1. `npm install`
2. `npm run start:dev`
